**Marketing strategy and thinking:**

- Value proposition and positioning
- Messaging frameworks
- Content strategy (blog, social, etc.)
- Pricing strategy and philosophy
- Copy guidelines (CTAs, headlines, etc.)
- Marketing planning docs

**What does NOT go here:**

- ❌ Tactical marketing tasks → GitHub Issues
- ❌ Roadmaps/timelines → GitHub Projects
- ❌ User requirements → docs/user/
- ❌ Technical implementation → docs/tech/
